# Web Icons for WidMate

## Required Icon Sizes:
- Icon-192.png (192x192px)
- Icon-512.png (512x512px)
- favicon.ico (32x32px)

## Generation Instructions:
1. Use the SVG file from assets/icons/app_icon.svg
2. Export as PNG in the required sizes
3. Place in web/icons/ directory
4. Update web/index.html to reference the icons

## Icon Usage:
- Icon-192.png: Used for PWA installation
- Icon-512.png: Used for PWA splash screen
- favicon.ico: Used for browser tab icon
